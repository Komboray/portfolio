INTRODUCTION TO SOCKETS IN PYTHON


Sockets in python language are very important because they habour for different functionalities such as communication between servers and clients.

Examples of Sockets in python include:

    1. UDP Sockets User datagram protocol
    2. TCP Sockets Transmission Control Protocol

In this case, we are going to be analyzing the TCP Server
    TCP Servers require an established connection before transmitting data
    Can sequence data (send in a specific order)
    Can retransmit data if packets fail to arrive
    Delivery is guaranteed
    Thorough error-checking guarantees data arrives in its intended state
    Not supported

TCP is best for:

    Email or texting
    File transfer
    Web browsing

Below is an example of a TCP Connection with the server side

            
        import socket
        #will be used to create an effiecient link with the server socket to plug us in
        
        #if the client is in the local area network, you just connect to the server ip address
        #Otherwise we need the ip address the public ip adress if the device is outside the LAN
        
        # ip address
        HOST = "192.168.106.1"
        PORT = 9098
        server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        server.bind((HOST,PORT))
        server.listen(1)
        
        while True:
            #we accept the message
            socket_mess,address = server.accept()
            print(f"You are connected to the following address{address}")
            message = input("Enter the message to tell the client what to do!")
            #we want to send the message to the client
            socket_mess.send(message.encode('utf-8'))
            #we want to receive what the client is going to say
            clientMess = socket_mess.recv(1024)
            print(f"The Client message is:{clientMess.decode('utf-8')}")
            socket_mess.close()
        #close the server
        # server.close()
